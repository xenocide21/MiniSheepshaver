MiniSheepshaver Configuration Tutorials


Here I will provide photos and information on how to install and configure Sheepshaver on both Windows and Macintosh. I am not much of a linux guru so you will have to sort that out on your own.


Windows Configuration

If you are a Windows user, follow this link to view the Windows Tutorial. The recommended system requirements for Sheepshaver is a Dual-Core Processor at 2.0GHz or better, 2048MB RAM (4096MB for Windows 7+) and Windows XP to Windows 10. The process is similar for Mac OS X 10.7 to macOS 10.14, however there are some key differences to installing Sheepshaver on macOS, thus there will be a separate tutorial for Mac users provided.
