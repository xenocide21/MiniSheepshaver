Welcome to the downloads section. Here you will find important oberin client updates, new feature releases, MiniSheepshaver builds, and additional content for MiniSheepshaver and MiniSheepshaverUltra.

MiniSheepshaver Builds


  • MiniSheepshaver v6.0
  • MiniSheepshaver v6.0 Image Only

    This is the lightest build of MiniSheepshaver, only bare essentials here, and a couple of useful programs you might need to enhance your oberin experience. If you already have Sheepshaver set up, you can download the image only, add it to your Sheepshaver directory, and set your config to boot from the new image file.

  • MiniSheepshaverUltra v1.0
  • MiniSheepshaverUltra v1.0 Image Only

    This build of MiniSheepshaver is quite a bit larger in size than MiniSheepshaver v6.0 however it is jam packed with a bunch of utilities, larger disk image for additional storage space, and can do much more than the minimalist MiniSheepshaver v6.0. Please take note only MiniSheepshaverUltra has a built in web browser. To maintain minimalist size for v6.0 a web browser has been omitted from the build.
